# Liste de Tâches 

Ceci est un petit projet pour gérer une liste de tâches en JavaScript, HTML et CSS.

## Fonctionnalités

- Ajouter une tâche dans la liste
- Voir les tâches sous forme de cartes (quadro)
- Prévenir si le champ est vide

## Installation

1. Cloner le projet ou télécharger les fichiers
2. Ouvrir `index.html` dans votre navigateur
3. Commencer à ajouter des tâches
